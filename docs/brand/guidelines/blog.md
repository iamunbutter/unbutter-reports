# 네이버 블로그 가이드

## 콘텐츠 방향

### 목적
- SEO 최적화
- 구매 전환
- 정보 제공

### 타겟
- 구매 고민 중인 고객
- 바디버터 검색자
- 비교/후기 찾는 사람

## 포스팅 구조

### 제목
```
[키워드] + [특징] + [후기/비교]

예시:
- 끈적이지 않는 바디버터 추천 | 언버터 핸디버터 후기
- 레몬향 바디버터 | 언버터 버터루틴 1개월 사용 후기
- 휴대용 바디버터 비교 | 직장인 필수템 언버터
```

### SEO 키워드 배치
- 제목에 1회
- 첫 문단에 1회
- 중간에 2-3회 자연스럽게
- 소제목에 변형 포함

### 본문 구성
```
1. 도입 (공감)
   - 고객 페인 포인트
   - 왜 찾게 되었는지

2. 제품 소개
   - 언버터 브랜드
   - 제품 특징
   - 사용감

3. 실제 사용 후기
   - 텍스처
   - 향
   - 지속력
   - 휴대성

4. 장단점
   - 솔직하게
   - 근거 있게

5. 추천 대상
   - 이런 사람에게 추천
   - 구매 링크 (자연스럽게)
```

## 이미지 최적화

### 개수
- 10-15장 적정
- 텍스트 설명 사이사이 배치

### 구성
- 제품 전체샷
- 텍스처 클로즈업
- 사용 장면
- 비교 이미지
- 패키지 디테일

### Alt 텍스트
- 키워드 포함
- 이미지 설명 정확히

## 키워드 전략

### 메인 키워드
- 바디버터
- 바디밤
- 바디크림
- 바디 앤 핸드크림

### 롱테일 키워드
- 끈적이지 않는 바디버터
- 휴대용 바디버터
- 레몬향 바디버터
- 비건 바디버터
- 민감성 피부 바디버터

### 연관 키워드
- 데일리 보습템
- 직장인 바디케어
- 핸디버터 추천

## 작성 톤

### 주인장 → 버터팬
- 친근한 존댓말 ("~해요", "~하세요")
- 솔직하고 진솔하게
- 과정과 고민 공유
- 함께 만들어가는 느낌

### 스토리텔링 중심
- 제품 뒤 이야기
- 샘플 테스트 과정
- 디테일한 고민
- 주인장의 일상

### 예시 (실제 블로그 톤)
```
✅ 좋음:
"두 달 간 7개가 넘는 샘플을 반려시키며
머릿속에 있는 제품을 실물로 짠! 구현해 내는 것이 
얼마나 어려운 일인지 새삼 깨달았습니다"

"제법 버터 같지요?"

"많이 많이 괴롭혀주세요"

"늦지 않게 또 글 적을게요 안녕 💛🤎"

❌ 나쁨:
"완전 대박! 이거 진짜 최고예요!
여러분 지금 바로 사세요!"

"혁신적인 포뮬러로 피부 장벽을 강화합니다"
```

### 말투 특징
- "~지요?", "~하네요", "~더라고요"
- "짠!", "그만큼 진심이고 간절하다는 의미이기도 하겠죠?"
- 괄호로 속마음 표현
- 이모지: 🧈 💛 🤎

## 링크 전략

### 내부 링크
- 관련 포스팅 연결
- 시리즈 포스팅 구성

### 외부 링크
- 공식몰 (자연스럽게)
- 구매 후기 (신뢰도)

## 업로드 주기

### 빈도
- 주 2-3회

### 요일
- 월, 수, 금 (검색 활발)
- 오전 10-11시 (점심 전 검색)

## 시리즈 기획

### 예시
1. 언버터 첫 구매 후기
2. 1주일 사용 후기
3. 1개월 장기 사용 후기
4. 다른 바디버터 비교
5. 계절별 사용팁

## 금지 사항

❌ 키워드 과도하게 반복
❌ 광고 표시 누락
❌ 과장된 효능 주장
❌ 타 브랜드 비하
❌ 허위 후기
