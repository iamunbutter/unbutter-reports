# 인스타그램 가이드

## 콘텐츠 방향

### 핵심 메시지
"매일 쓰고 싶은 버터루틴"

### 감성
- 베이커리 무드
- 일상의 작은 행복
- 루틴의 아름다움

## 피드 규칙

### 비주얼
- 브랜드 컬러 중심 (#fff1b5, #43302e)
- 제품 + 소품 조합
- 자연광 선호
- 베이커리 소품 활용

### 구도
- 플랫레이
- 핸드인샷
- 루틴 장면
- 데스크/화장대 연출

## 캡션 작성

### 길이
- 2-3문장 (간결하게)
- 핵심만 전달

### 톤
- 친근한 반말
- 이모지 적절히 (🧈, 💛, ✨)
- 진솔하게

### 구조
```
[후크 문장]
[제품/사용감 설명]
[행동 유도 or 공감 유도]

#해시태그
```

### 예시
```
아침마다 바르는 버터루틴 🧈

끈적이지 않아서
바로 옷 입고 나갈 수 있어요

30g 핸디버터, 가방에 쏙 💛

#언버터 #바디버터 #데일리루틴
```

## 릴스 전략

### 길이
- 7-15초 (짧고 강렬하게)

### 구성
```
0-3초: 후크 (시선 잡기)
3-10초: 내용 (사용 장면)
10-15초: CTA (행동 유도)
```

### 자막
- 한 화면에 5-7자
- 읽기 쉬운 폰트
- 브랜드 컬러 사용

### 음악
- 트렌디한 사운드
- 베이커리/카페 무드
- 라이센스 확인 필수

## 스토리 활용

### 일상 공유
- 루틴 인증
- 사용 장면
- 패키지 언박싱

### 소통
- 질문 스티커
- 투표
- 퀴즈

### 하이라이트
- 제품 소개
- 고객 후기
- 사용법
- FAQ

## 해시태그 전략

### 개수
- 10-15개 (너무 많지 않게)

### 구성
```
브랜드 (3개)
#언버터 #unBUTTER #버터루틴

제품 (3-4개)
#바디버터 #바디밤 #핸디버터 #홈버터

감성 (3-4개)
#데일리루틴 #바디케어 #보습템 #시그니처향

타겟 (2-3개)
#직장인템 #데스크테리어 #화장대인테리어
```
