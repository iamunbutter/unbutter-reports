# 메타 광고 콘텐츠 프레임워크 — "영화만들기"

> 출처: 자사몰 사관학교 · 김성현 CMO (스킨스쿨, 자사몰 누적 500억)

---

## 핵심 공식

### 메타 경매 공식
```
광고비 × 추산행동률 + 광고품질 = 총가치
         (고객이 살 확률)   (고객의 콘텐츠 반응도)
```

- 총가치(Total Value)가 높은 광고 → 예산 추가 시에도 좋은 유저·좋은 순간에 노출
- ROAS 하락 적고 확장성 우수

### 스노우볼 효과

**악순환:**
성과 나쁜 광고 OFF → 추산행동률 하락 → CPM 상승 → 좋은 고객 입찰 실패 → 성과 악화 반복

**선순환:**
뾰족한 콘텐츠 → 머신러닝(기다림) → 성과 개선 → CPM 하락 → 좋은 고객 입찰 → 추산행동률 상승

**결론: 그 어떤 '광고세팅'도 '좋은 콘텐츠'를 이길 수 없다.**

## 프레임워크: 영화만들기

### 4단계 프로세스

| 단계 | 내용 |
|------|------|
| 1. 주인공 설정 | 대표가 바라보는 고객/제품 + 고객이 바라보는 제품 → 페르소나 |
| 2. 장면 만들기 | 고객의 '문제'와 '해결방법' 상황 구현 |
| 3. 장면 검증 | 고객가치 기반 콘텐츠 제작 + 캠페인 세팅 + 성과 테스트 |
| 4. 배움/적용 | 회고 및 판매캠페인 진행 |

---

## 주인공 정의

### 주인공 = 작가시점 ∩ 고객시점

```
주인공 = (작가의 강점·철학·서사) ∩ (고객의 환경·익숙함·기대)
```

브랜드는 '나만의 색'으로 차별화하고, 고객의 익숙함으로 공감시키며, 그 둘의 교집합에서 판매가 폭발한다.

### 작가시점 (대표가 정의하는 고객과 제품)

핵심 질문:
- 왜 이 제품을 만들었나?
- 이 제품은 어떤 방법으로 고객의 문제를 해결하나?
- 제품의 강점은?
- 고객은 어떤 사람인가?
- 어떤 경로로 구매하나?
- 구매 이유 / 구매 고민은?

→ **작가시점 13가지 질문 템플릿 작성 필요**

### 고객시점 (VOC — Voice of Customer)

수집 방법:
1. 설문조사
2. 1:1 인터뷰
3. 온라인 리뷰 분석
4. SNS/커뮤니티 조사
5. 사용자 테스트 (User Testing)

### 인터뷰 핵심 TIP

- 듣는 것이 목적. 예시를 먼저 주지 마라.
- **문제점을 말하게 하라, 해결책을 묻지 마라.**
- 경험의 과정을 질문하라 → 진실에 가까운 답변
- "다른 고객에게 추천한다면 한 문장으로?" → 광고 카피 금맥
- Yes/No 질문 피하기
- 이름을 불러주기

## 구매 결정 요인

| 요인 | 설명 |
|------|------|
| 가치 (Unique Value) | 고객에게 전달되는 독창적 혜택 |
| 효용 (Utility) | 제품/서비스 사용 실질적 만족도, 문제 해결 정도 |

**핵심:** "간편하고 안전한 계좌이체 서비스" vs "10초만에 끝나는 빠른 송금"
→ 후자가 끌리는 이유: 고객의 눈높이에서 가치를 표현했기 때문

## 메타 AI와 주인공 정의의 관계

메타 AI는 실시간으로 소비자 선호도를 기억하고 추천한다.
→ 소비자가 선호할 거라 판단했는데 반응하지 않으면, 광고는 아예 기회에서 배제된다.
→ 따라서 주인공 정의가 정확해야 메타 AI가 올바른 고객에게 노출할 수 있다.
