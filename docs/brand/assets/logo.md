# 로고 가이드

## 메인 로고

**변형 로고 (주 사용)**

`un` + `BUTTER`

- 베이커리 감성 극대화
- 패키지/SNS/이벤트 메인

## 브랜드 컬러

### Primary
- **#fff1b5** (Soft Butter) - 부드러운 노란빛
- **#43302e** (Cocoa Latte) - 따뜻한 브라운

### 사용 원칙
- 배경: Soft Butter (#fff1b5)
- 텍스트/강조: Cocoa Latte (#43302e)
- 조합: 따뜻하고 베이커리 느낌

## 로고 사용 영역

### 변형 로고 사용
- 용기 / 단상자 / 키트
- 인스타그램 / 릴스
- 팝업스토어
- 이벤트 페이지
- 굿즈

→ **감성·바이럴·차별화 담당**

## 금지 사항

❌ 로고 비율 변경
❌ 로고 색상 임의 변경
❌ 로고에 그림자/효과 추가
❌ 배경과 대비 낮은 색 사용
