## 광고 1편 — 로션 정착 실패자

### 광고 제목 (내부용)
`ADV-01 | "로션 정착 실패자"` — v2 주인공 시선

### 리라이팅 근거
기존 훅 "바디로션, 또 3번 쓰고 포기하셨죠?"는 브랜드가 고객을 진단하는 시선. 서윤씨는 광고 티 나는 콘텐츠를 곧바로 스크롤한다. 서윤씨 본인이 친구에게 털어놓듯 말하는 언어로 전환.

### 훅 (0~3초)

| 요소 | 내용 |
|------|------|
| 장면 | 서윤씨 욕실. 선반 위 반쯤 쓴 바디로션 병들. 카메라가 병을 천천히 훑다가 서윤씨 손으로 이동. 핸디버터를 꺼내 손바닥에 올리는 장면으로 자연스럽게 전환. |
| 자막/카피 | **"바디로션은 왜 항상 선반 위에서 굳어가지..."** |
| 연출 포인트 | 혼잣말 톤. 자막 폰트 손글씨 느낌 or 심플 고딕. 배경음 없이 시작 — 일상 사운드(수도 소리 미약하게) 깔림. |

### 본문 (3~15초)

| 시간 | 장면 설명 | 자막 / 나레이션 |
|------|-----------|----------------|
| 3~5초 | 서윤씨가 바디로션을 펌프하는 클로즈업. 손에 묵직하게 쏟아지는 텍스처. 잠깐 망설이는 손. | *"무거워서, 바르고 나서 침구에 묻을 것 같아서."* |
| 5~8초 | 로션 병을 다시 선반에 올려두는 손. 불을 끄는 장면으로 이어짐. | *"오늘도 그냥 자자 — 이 루틴이 반복됐어요."* |
| 8~11초 | 다음 씬. 같은 욕실, 이번엔 핸디버터를 꺼내 손등에 소량 올리는 클로즈업. 스르르 녹아드는 텍스처. | *"이건 달랐어요. 바르자마자 흡수되고,"* |
| 11~14초 | 바른 손을 흔들어 보이다가 자연스럽게 수건 집어 드는 장면. 끈적임 없음이 행동으로 증명됨. | *"끈적임이 없어서 바로 잘 수 있거든요."* |

### CTA (15~20초)

| 요소 | 내용 |
|------|------|
| 장면 | 침대 협탁 위에 핸디버터 하나 놓인 클로즈업. 정돈된 침구, 간접조명. 서윤씨의 실제 침실 같은 온도감. |
| 자막 | **"자려고 손 뻗었다가 자연스럽게 바르게 돼요."** |
| 서브 카피 | 언버터 핸디버터 |
| 하단 CTA | 써보기 |

### 타겟 오디언스
- 여성 24~35세, 수도권 거주 직장인
- 관심사: 스킨케어 루틴, 미니멀 홈케어, 슬로우 라이빙, 셀프케어
- 행동: 최근 60일 내 바디케어 또는 핸드크림 검색 · 구매 이력
- 제외: 이미 구매 완료 전환 고객

### BGM
무음으로 시작 (0~5초). 5초부터 미니멀 Lo-fi 피아노 페이드인. 볼륨 낮게 유지 — 나레이션이 주인공.

### 촬영 노트
- **공간**: 29세 직장인 여성의 실제 욕실 + 침실. 과한 스타일링 금지 — 칫솔, 세안폼, 일반 소품 자연스럽게 노출.
- **앵글**: 욕실 선반 수평 클로즈업 / 손 텍스처 매크로 / 침대 협탁 탑뷰
- **조명**: 욕실 씬 쿨톤 형광 → 언버터 등장 이후 협탁 씬 웜 간접조명으로 자연스럽게 전환
- **모델 디렉션**: 카메라 의식하지 않는 자연스러운 동작. 표정 연기 최소화 — 손 동작으로 감정 전달.
- **소품**: 유사 바디로션 병(레이블 테이프로 가리기), 핸디버터, 수건, 침구(리넨 톤)

### KPI

| 지표 | 목표 |
|------|------|
| Hook Rate (3초) | 42%+ |
| 완주율 (15초+) | 28%+ |
| CTR | 1.5%+ |
| CPC | 300원 이하 |

## 광고 2편 — 얼굴과 몸의 불균형

### 광고 제목 (내부용)
`ADV-02 | "얼굴과 몸의 불균형"` — v2 주인공 시선

### 리라이팅 근거
기존 훅 "얼굴엔 6단계, 몸엔 0단계?"는 수치 대비로 브랜드 시각. 서윤씨는 성분 설명보다 사용 장면을 먼저 본다. 그리고 스킨케어 중 잠깐 팔꿈치를 내려다보는 서윤씨 본인의 찰나로 전환. 또한 기존 본문에 "시어버터 15%"라는 성분 언급이 있었는데 이를 제거하고 사용 장면으로 대체.

### 훅 (0~3초)

| 요소 | 내용 |
|------|------|
| 장면 | 화장대 앞 서윤씨. 얼굴에 에센스를 바르다가 무심코 자신의 팔꿈치를 내려다보는 클로즈업. 잠깐의 정적. |
| 자막/카피 | **"얼굴엔 공들이는데 팔꿈치는 언제 마지막으로 봤지?"** |
| 연출 포인트 | 자막 등장 전 0.5초 정도 장면만 흐름 — 시청자가 먼저 장면을 인식한 후 자막이 그 감정을 언어화. |

### 본문 (3~15초)

| 시간 | 장면 설명 | 자막 / 나레이션 |
|------|-----------|----------------|
| 3~6초 | 얼굴 루틴 클로즈업 (스킨 → 에센스 → 크림) 빠른 컷 전환. 꼼꼼한 손길. | *"스킨, 에센스, 크림, 선크림..."* |
| 6~9초 | 같은 손이 멈추고, 팔꿈치와 손등을 번갈아 보는 장면. 건조한 피부 클로즈업. | *"그런데 팔꿈치는 언제부터 이랬지?"* |
| 9~12초 | 화장대 한 켠에서 핸디버터를 꺼내 팔꿈치에 부드럽게 펴 바르는 장면. 스킨케어 제품들 사이에 자연스럽게 놓인 핸디버터. | *"스킨케어 루틴 사이에 30초면 돼요."* |
| 12~14초 | 매끄럽게 정돈된 팔꿈치 피부 클로즈업. 서윤씨가 소매를 내리고 가방을 드는 장면으로 전환. | *"바르고 바로 소매 입어도 되고요."* |

### CTA (15~20초)

| 요소 | 내용 |
|------|------|
| 장면 | 화장대 위 스킨케어 제품들 사이에 핸디버터가 자연스럽게 자리 잡은 플랫레이. 인스타 피드 같은 구도. |
| 자막 | **"화장대 루틴에 하나 더 자리 생겼어요."** |
| 하단 CTA | 언버터 바디버터 알아보기 |

### 타겟 오디언스
- 여성 23~38세
- 관심사: 페이셜 스킨케어, K-뷰티, 홈케어 루틴, 화장대 꾸미기, 데스크테리어
- 유사타겟: 기존 구매자 기반 2% LAL
- 행동: 스킨케어 제품 페이지 조회 이력 보유자

### BGM
아침 루틴 느낌의 미니멀 어쿠스틱. 훅(0~3초)은 거의 무음 — 장면 자체가 훅. 9초부터 가볍게 페이드인. BPM 88~95.

### 촬영 노트
- **공간**: 서윤씨 스타일의 화장대 (깔끔하되 과하지 않음. 탬버린즈, 논픽션 류 제품이 자연스럽게 섞인 화장대).
- **앵글**: 화장대 거울 반사 활용한 미디엄샷 + 팔꿈치 클로즈업 + 탑뷰 플랫레이
- **조명**: 아침 자연광 기반. 화장대 조명 보조. 쿨하지 않게 — 아이보리 웜톤 유지.
- **모델 디렉션**: 얼굴 루틴 장면은 능숙하게 빠르게. 팔꿈치 내려다보는 장면은 연기 없이 자연스러운 무심한 시선으로.
- **편집**: 얼굴 루틴과 팔꿈치 장면 대비는 빠른 컷. 핸디버터 등장 이후는 슬로우.

### KPI

| 지표 | 목표 |
|------|------|
| Hook Rate (3초) | 45%+ |
| 완주율 | 28%+ |
| CTR | 1.8%+ |
| 브랜드 인지도 리프트 | 메타 Brand Lift 조사 활용 |

## 광고 3편 — 버터를 몸에?

### 광고 제목 (내부용)
`ADV-03 | "버터를 몸에?"` — v2 주인공 시선

### 리라이팅 근거
기존 버전은 인물이 카메라에 직접 설명하는 구조. 효과적이지만 서윤씨는 광고 티 나는 콘텐츠를 스크롤한다. 훅은 유지하되, 본문을 서지은(UCM 페르소나)의 실제 하루 장면으로 전환. 나레이션 대신 장면이 말하게.

### 훅 (0~3초)

| 요소 | 내용 |
|------|------|
| 장면 | 카페 테이블. 노트북 옆에 핸디버터가 놓여 있다. 지나가는 친구가 집어 들며 "이게 뭐야?" 하는 표정. 서지은이 씩 웃는 클로즈업. |
| 자막/카피 | **"몸에 바르는 버터예요."** |
| 연출 포인트 | 대화 톤. 친구에게 설명하듯 자연스럽게. 자막은 서지은의 말처럼 등장. |

### 본문 (3~15초)

| 시간 | 장면 설명 | 자막 / 나레이션 |
|------|-----------|----------------|
| 3~6초 | 서지은이 핸디버터를 손등에 소량 올리는 클로즈업. 손 위에서 녹아드는 텍스처. 친구가 옆에서 지켜보는 장면. | *"손등에 이렇게 올려서 문지르면..."* |
| 6~9초 | 바른 직후 손가락을 벌려 끈적임이 없음을 보여주는 장면. 노트북 키보드를 그대로 치는 손. | *"바로 키보드 쳐도 돼요. 끈적이지 않아서."* |
| 9~12초 | 친구가 직접 손목에 발라보는 장면. 향을 맡는 클로즈업. 눈이 동그래지는 표정. | *"레몬 버터 향인데, 은은해서 부담 없어요."* |
| 12~14초 | 서지은의 가방 안 핸디버터 클로즈업. 서지은이 가방을 닫으며 "항상 갖고 다녀" 하는 듯한 제스처. | *"어디든 들고 다니거든요. 건조할 때마다 꺼내서."* |

### CTA (15~20초)

| 요소 | 내용 |
|------|------|
| 장면 | 카페 테이블 위, 핸디버터와 노트북이 함께 놓인 자연스러운 데스크테리어 샷. 인스타 스토리 같은 구도. |
| 자막 | **"한번 써보면 설명 안 해도 알아요."** |
| 하단 CTA | 핸디버터 살펴보기 |

### 타겟 오디언스
- 여성 20~34세 / 남성 보조 타겟 20~30세
- 관심사: 뷰티 신제품, 카페 브이로그, 데스크테리어, 세미 인플루언서 팔로우
- 이 편은 신규 인지 확장용 — 넓은 타겟, CPM 최적화 집행
- 행동: 뷰티/라이프스타일 릴스 최근 30일 내 시청 이력

### BGM
친구끼리 카페에서 대화하는 온도감. 가볍고 자연스러운 Lo-fi 팝. 훅(0~3초)은 카페 앰비언트 사운드(배경 소음 낮게) + BGM 동시 진입. BPM 100~108.

### 촬영 노트
- **공간**: 실제 카페. 스튜디오 세트 금지. 자연광 들어오는 창가 자리 우선.
- **앵글**: 정면 미디엄샷 (두 인물) + 손 클로즈업 매크로 + 가방 안 클로즈업
- **조명**: 카페 자연광 위주. 보조 반사판 사용 가능하나 인위적 느낌 배제.
- **모델 디렉션**: 두 인물 모두 연기 최소화. 실제 대화하는 것처럼. 카메라 인식하지 않는 방향으로.
- **소품**: 핸디버터, 카페 음료(아이스라떼 류), 맥북 or 노트북, 토트백, 에어팟
- **편집**: 점프컷으로 빠른 전개. 향 맡는 장면만 0.5초 슬로우모션.

### KPI

| 지표 | 목표 |
|------|------|
| Hook Rate (3초) | 50%+ (궁금증 유발) |
| 완주율 | 35%+ |
| CTR | 2.0%+ |
| 공유/저장 수 | 타 편 대비 +30% |

## 광고 4편 — 실패 없는 선물

### 광고 제목 (내부용)
`ADV-04 | "실패 없는 선물"` — v2 주인공 시선

### 리라이팅 근거
기존 버전의 훅 "줄 때마다 좋아했던 그 선물"은 브랜드가 보증하는 느낌. 서윤씨 관점에서 선물을 주는 이유는 따로 있다 — "내가 좋아하는 걸 줘서 실패가 없다". 서윤씨가 먼저 쓰다가 선물하게 된 흐름으로 재구성. 기존 본문의 "비건 성분" 표현은 성분 설명이므로 제거.

### 훅 (0~3초)

| 요소 | 내용 |
|------|------|
| 장면 | 서윤씨가 화장대에서 핸디버터를 쓰는 장면. 컷 전환. 포장된 같은 제품을 친구에게 건네는 장면. |
| 자막/카피 | **"내가 매일 쓰는 거라 선물했어요."** |

## 광고 5편 — 화장대 앞 바디 보습

### 광고 제목 (내부용)
`ADV-05 | "화장대 앞 바디 보습"` — v2 주인공 시선

### 리라이팅 근거
기존 훅 "얼굴만 챙기고 출근하시나요?"는 브랜드가 고객에게 묻는 형식 — 지적받는 느낌. 서윤씨는 그런 광고를 스크롤한다. 대신 서윤씨가 본인 루틴 안에서 자연스럽게 핸디버터를 끼워 넣는 장면 자체를 훅으로. 기존 CTA "아침 루틴의 마지막 단계"는 지시형이므로 서윤씨의 말로 교체.

### 훅 (0~3초)

| 요소 | 내용 |
|------|------|
| 장면 | 서윤씨가 아침 화장대 앞에서 스킨케어를 마치고, 핸디버터를 집어 들어 손목에 바르는 자연스러운 장면. 어떤 설명도 없이 행동만. |
| 자막/카피 | **"스킨케어 마지막에 이것도 바르거든요."** |
| 연출 포인트 | 브이로그 같은 핸드헬드 느낌. 삼각대 금지. 일기처럼 담담하게. |

### 본문 (3~15초)

| 시간 | 장면 설명 | 자막 / 나레이션 |
|------|-----------|----------------|
| 3~6초 | 화장대 앞, 스킨케어 제품들 사이에서 핸디버터를 꺼내는 손. 다른 제품들과 나란히 놓인 핸디버터. | *"얼굴이랑 같이 손목, 팔꿈치에 한 번씩."* |
| 6~9초 | 손목에 소량 올리고 문지르는 클로즈업. 흡수되는 텍스처. | *"끈적이지 않아서 바로 옷 입어도 돼요."* |
| 9~12초 | 화장이 완성된 서윤씨가 핸디버터를 가방에 넣는 장면. 아침 햇살이 드는 현관. | *"나가서도 건조하면 꺼내 쓰니까."* |
| 12~14초 | 지하철 안. 서윤씨가 손목에 살짝 더 발라보는 장면. 자연스러운 출퇴근 루틴. | *"하루에 두세 번, 어느새 습관이 됐어요."* |

### CTA (15~20초)

| 요소 | 내용 |
|------|------|
| 장면 | 화장대 스킨케어 제품들 사이에 핸디버터가 자연스럽게 놓인 플랫레이. 인스타 피드 감성. |
| 자막 | **"없으면 조금 아쉬운 루틴템이 됐어요."** |
| 하단 CTA | 핸디버터 살펴보기 |

### 타겟 오디언스
- 직장인 여성 24~36세, 수도권 거주
- 행동: 출퇴근 시간대 (오전 7~9시, 저녁 7~9시) 활성 유저 우선 게재
- 관심사: 아침 루틴, 직장인 브이로그, 스킨케어, 데스크테리어
- 지역: 서울·경기·인천 직장 밀집 지역

### BGM
아침 느낌의 잔잔한 재즈 피아노. 과하지 않게. BPM 95~105. 지하철 장면에서는 배경 지하철 소음을 낮게 유지하며 BGM과 혼합.

### 촬영 노트
- **공간**: 실제 직장인 여성 아파트 화장대 + 지하철 내부. 세트 촬영 최소화.
- **앵글**: 화장대 거울 반사 활용한 핸드헬드 사이드샷 + 손 클로즈업 + 지하철 창가 샷
- **조명**: 아침 자연광 (커튼 반만 친 상태). 화장대 조명 보조. 지하철 씬은 자연광.
- **모델 디렉션**: 카메라를 의식하지 않는 방향. 브이로그 촬영하는 것처럼 셀프캠 앵글 활용 가능.
- **소품**: 화장대(스킨케어 제품 3~5종, 핸디버터), 출근룩(캐주얼 오피스 스타일), 토트백, 에어팟
- **편집**: 아침 → 지하철 흐름이 자연스럽게 이어지도록. 빠른 컷이지만 브이로그처럼 여유 있게.
