# 언버터 마케팅 에이전트 팀 구조

## 🎯 목표
- SNS 콘텐츠 기획 자동화 (인스타, 블로그, 유튜브)
- 채널별 성과 분석 (메타 광고, 카페24 매출)
- 마케팅 워크플로우 자동화

---

## 👥 에이전트 팀 구성

### 1. 📝 콘텐츠 기획 에이전트 (Content Planner)
**역할:** 브랜드 정체성에 맞는 SNS 콘텐츠 아이디어 생성

**입력:**
- 브랜드 가이드 (ICP, 페르소나, UCM)
- 트렌드 키워드
- 경쟁사 분석

**출력:**
- 인스타그램 피드/릴스 콘텐츠 아이디어
- 블로그 포스트 주제
- 유튜브 숏폼 스크립트

**자동화 포인트:**
- 주간 콘텐츠 캘린더 자동 생성
- 계절/이벤트 기반 콘텐츠 제안
- 페르소나별 메시지 커스터마이징

---

### 2. ✍️ 카피라이터 에이전트 (Copywriter)
**역할:** 채널별 최적화된 카피 작성

**입력:**
- 콘텐츠 기획서
- 브랜드 톤앤매너 (캐주얼+프로페셔널)
- 채널별 규격 (인스타 해시태그, 블로그 SEO)

**출력:**
- 인스타그램 캡션 (해시태그 포함)
- 블로그 본문 (SEO 최적화)
- 광고 카피 (메타 광고용)

**자동화 포인트:**
- 베이커리 컨셉 키워드 자동 삽입
- 감성적 표현 + 제품 정보 균형
- A/B 테스트용 카피 변형 생성

---

### 3. 🎨 비주얼 디렉터 에이전트 (Visual Director)
**역할:** 촬영 콘셉트 및 디자인 가이드 제시

**입력:**
- 콘텐츠 기획서
- 브랜드 컬러 (#fff1b5, #43302e)
- 레퍼런스 이미지

**출력:**
- 촬영 콘셉트 및 소품 리스트
- 이미지 편집 가이드
- 릴스/숏폼 스토리보드

**자동화 포인트:**
- 무드보드 자동 생성 (AI 이미지 활용)
- 계절/트렌드 기반 비주얼 방향성 제안
- 상세페이지 섹션별 이미지 요구사항 정리

---

### 4. 📊 성과 분석 에이전트 (Analytics Agent)
**역할:** 채널별 데이터 수집 및 인사이트 도출

**입력:**
- 메타 광고 데이터
- 카페24 매출 데이터
- 인스타그램 인사이트
- 네이버 키워드 분석

**출력:**
- 주간/월간 성과 리포트
- 고성과 콘텐츠 분석
- 개선 포인트 제안

**자동화 포인트:**
- 데이터 자동 수집 (API 연동)
- 대시보드 업데이트
- 성과 하락 시 알림

---

### 5. 🤖 CRM 자동화 에이전트 (CRM Automation)
**역할:** 고객 여정별 메시지 자동 발송

**입력:**
- 구매 데이터 (카페24)
- 고객 여정 맵
- 메시지 템플릿

**출력:**
- 구매 후 1일/1주/4주차 메시지
- 장바구니 이탈 메시지
- 재구매 유도 메시지

**자동화 포인트:**
- 트리거 기반 메시지 발송
- 고객 세그먼트별 메시지 커스터마이징
- 응답률 추적 및 개선

---

### 6. 🎯 광고 매니저 에이전트 (Ad Manager)
**역할:** 메타 광고 캠페인 기획 및 최적화

**입력:**
- 타겟 페르소나
- 광고 예산
- 성과 데이터

**출력:**
- 광고 세트 구성안
- 타겟팅 전략
- 예산 배분 제안

**자동화 포인트:**
- 고성과 소재 기반 신규 광고 생성
- 예산 자동 조정
- A/B 테스트 실행

---

## 🔄 워크플로우

```
1. 콘텐츠 기획 에이전트 → 주간 콘텐츠 캘린더 생성
   ↓
2. 카피라이터 + 비주얼 디렉터 → 콘텐츠 제작 가이드 생성
   ↓
3. (수동) 실제 촬영 및 편집
   ↓
4. (수동) SNS 업로드
   ↓
5. 성과 분석 에이전트 → 데이터 수집 및 리포트
   ↓
6. 광고 매니저 → 고성과 콘텐츠 기반 광고 집행
   ↓
7. CRM 자동화 → 구매 고객 메시지 발송
```

## 📂 필요한 데이터 및 도구

**브랜드 자료 (완료)**
- ✅ ICP 정의서
- ✅ 페르소나 보드
- ✅ UCM
- ✅ 제품 개발 정의서
- ✅ 상세페이지 기획안
- ✅ 브랜드 로고 및 컬러

**연동 필요**
- 메타 광고 API
- 카페24 API
- 인스타그램 인사이트 API
- 네이버 키워드 도구

**추가 작업**
- 콘텐츠 템플릿 정리 (캡션, 해시태그, 스크립트)
- CRM 메시지 템플릿 작성
- 브랜드 톤앤매너 가이드 문서화

---

## 🚀 1단계 실행 계획

**이번 주 (테스트 런)**
1. 콘텐츠 기획 에이전트 구축
   - 1주일치 인스타 피드/릴스 아이디어 생성
   - 정인이 피드백 → 에이전트 개선
   
2. 카피라이터 에이전트 구축
   - 생성된 아이디어 기반 캡션 작성
   - 브랜드 톤 체크

**다음 주**
3. 비주얼 디렉터 에이전트 구축
4. 성과 분석 대시보드 설계

**이후**
5. CRM 자동화 구축
6. 광고 매니저 에이전트 구축

---

## 💡 참고사항
- 에이전트는 모두 수정 가능한 설정 파일로 관리
- 브랜드 정체성이 변하면 설정만 바꾸면 됨
- 각 에이전트 출력은 정인이 최종 승인 후 사용
