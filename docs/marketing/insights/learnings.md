# 학습한 인사이트 (Learnings)

_마케팅 과정에서 배운 교훈들_

## 런칭 전 인사이트

### 브랜드 포지셔닝
- 바디버터 시장은 "무겁다"는 선입견 존재
- → 끈적임 없음을 강조하는 게 핵심 차별점
- → 휴대성(핸디버터)으로 데일리 사용 강조

### 타겟 고객
- 20-30대 직장인 여성
- SNS 감성 소비 중요
- 패키지 디자인이 구매 결정에 큰 영향

### 채널 전략
- 인스타그램: 릴스 중심 (도달 넓히기)
- 블로그: SEO (구매 고민 단계 전환)
- 광고: 고성과 콘텐츠 기반 확장
