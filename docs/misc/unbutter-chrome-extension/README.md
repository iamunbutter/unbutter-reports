# 🧈 언버터 블로그 자동화 크롬 확장

네이버 블로그 자동 작성 도우미 (정인 전용)

## 설치 방법

1. **크롬 확장 프로그램 페이지 열기**
   - `chrome://extensions/` 주소창에 입력

2. **개발자 모드 활성화**
   - 우측 상단 "개발자 모드" 토글 켜기

3. **압축해제된 확장 프로그램을 로드합니다**
   - "압축해제된 확장 프로그램을 로드합니다" 클릭
   - `/Users/jeong-ing/.openclaw/workspace/unbutter-chrome-extension` 폴더 선택

4. **확장 프로그램 고정**
   - 툴바에서 확장 프로그램 아이콘 고정

## 사용 방법

1. **네이버 블로그 로그인**
   - `https://blog.naver.com/` 접속
   - 평소처럼 로그인

2. **글쓰기 페이지 열기**
   - "글쓰기" 버튼 클릭
   - 또는 `https://blog.naver.com/imunbutter/postwrite`

3. **확장 프로그램 실행**
   - 툴바의 🧈 아이콘 클릭
   - 제목 입력
   - 본문 입력 (줄바꿈 자동 처리)
   - "자동 작성하기" 버튼 클릭

4. **완료**
   - 자동으로 제목/본문 입력됨
   - 이미지는 수동 추가
   - 임시저장 또는 발행 버튼 클릭

## 장점

✅ **로그인 문제 없음** (직접 로그인)
✅ **봇 감지 회피** (실제 브라우저 세션 사용)
✅ **어디서나 사용** (크롬만 있으면 OK)
✅ **간단한 조작** (버튼 한 번)
✅ **안전** (로컬에서만 작동)

## 파일 구조

```
unbutter-chrome-extension/
├── manifest.json        # 확장 프로그램 설정
├── popup.html          # 팝업 UI
├── popup.js            # 팝업 로직
├── content.js          # 페이지 조작 스크립트
├── icons/              # 아이콘 (추후 추가)
└── README.md           # 이 파일
```
