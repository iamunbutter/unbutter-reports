# 톤앤매너 가이드

## 브랜드 보이스

**캐주얼하면서 프로페셔널**

- 친근하지만 신뢰감 있게
- 불필요한 존댓말보다 친근한 반말 (하지만 존중은 유지)
- "Great question!" 같은 형식적 멘트 지양
- 진심으로 도움 되는 정보 전달

## 언어 스타일

### ✅ 좋은 예시
- "매일 쓰고 싶은 향이에요"
- "가방에 쏙 들어가요"
- "끈적이지 않아서 바로 키보드 칠 수 있어요"
- "화장대에 두면 예쁘더라고요"

### ❌ 나쁜 예시
- "혁신적인 포뮬러로 피부 장벽을 강화합니다" (너무 딱딱)
- "지금 바로 구매하세요!" (강압적)
- "대박 이벤트!" (선정적)
- "최고의 바디버터" (과장)

## 채널별 톤

### 인스타그램 피드
- 간결하고 감각적
- 이모지 적절히 사용 (🧈, 💛, ✨)
- 2-3문장으로 핵심 전달
- 해시태그는 자연스럽게

### 인스타그램 릴스
- 자막: 짧고 읽기 쉽게
- 말투: 친구에게 말하듯
- 호흡: 빠르게 (숏폼 특성)

### 네이버 블로그
- 정보 중심이지만 읽기 편하게
- 단락 짧게, 이미지 많이
- SEO 키워드 자연스럽게 삽입
- 후기 톤: 진솔하게

### 상세페이지
- 설득력 있지만 과장 없이
- 근거 있는 주장
- 고객 관점 서술
- 전문 용어 최소화

## 금지 표현

❌ "혁신적", "최고", "대박"
❌ "지금 바로", "서둘러"
❌ "놓치면 후회"
❌ 과도한 이모지 (!!!, 😍😍😍)
❌ 전문 용어 남발

## 핵심 키워드

**반복 사용해도 좋은 단어들:**

- 버터루틴
- 은은한
- 끈적이지 않는
- 매일
- 시그니처 향
- 녹아드는
- 데일리
- 루틴
- 촉촉한
- 포근한

## 감성 포인트

### 베이커리 컨셉
- 갓 구운
- 달콤한
- 크림
- 타르트
- 레몬 제스트
- 버터 크러스트

### 루틴 강조
- 매일
- 하루의 시작
- 마무리
- 습관
- 지속
- 반복

### 감각 표현
- 녹아드는
- 스며드는
- 퍼지는
- 감싸는
- 밀착되는

## 이모지 사용 가이드

### 자주 사용
🧈 (버터) - 브랜드 시그니처
💛 (노란 하트) - 브랜드 컬러
✨ (반짝) - 특별함 강조

### 가끔 사용
🍋 (레몬) - 향 언급 시
🌿 (잎) - 자연/비건 강조
☁️ (구름) - 부드러움 표현

### 사용 금지
😍😘🤩 (과한 감정 표현)
💰💸 (돈 관련)
🔥💥 (선정적)

## 스토리텔링 가이드

**고객의 하루에 자연스럽게 녹아들기**

```
아침 7시, 출근 준비하면서
팔꿈치에 톡 바르고

오후 3시, 손이 건조할 때
책상에서 꺼내 바르고

밤 10시, 샤워 후
하루 마무리하며 바르고
```

→ 제품을 '사는 것'이 아니라 '일상에 스며드는 것'으로 표현
