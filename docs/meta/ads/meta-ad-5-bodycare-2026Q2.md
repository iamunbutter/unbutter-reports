## 광고 1편 — 바디로션 정착 못하는 이유

### 광고 제목 (내부용)
`ADV-01 | "로션 정착 실패자"`

### 훅 (0~3초)
| 요소 | 내용 |
|------|------|
| 장면 | 욕실 선반 위에 반만 쓰다 굳은 바디로션 3~4병 클로즈업. 먼지 살짝 앉아 있음. |
| 자막/카피 | **"바디로션, 또 3번 쓰고 포기하셨죠?"** |
| 연출 포인트 | 텍스트가 화면 중앙에 크고 담백하게 등장. 배경음 없이 조용히 시작 → 시청자 멈춤 유도 |

### 본문 (3~15초)

| 시간 | 장면 설명 | 자막 / 나레이션 |
|------|-----------|----------------|
| 3~5초 | 손에 바디로션 펌프 → 질퍽하게 나오는 텍스처 클로즈업 | *"무거워서, 끈적여서, 향이 강해서."* |
| 5~8초 | 인물이 로션 바른 손을 보며 찡그리는 표정 (핸드헬드, 자연스럽게) | *"결국 선반 위에 방치."* |
| 8~11초 | 언버터 핸디버터 등장. 손바닥 위에 올려놓고 피부에 스르르 녹아드는 텍스처 클로즈업 | *"언버터는 달라요."* |
| 11~14초 | 발랐는데도 손이 보송보송한 느낌 — 키보드 위에 손 올리는 장면 | *"끈적임 없이 녹아드는 버터 보습."* |

### CTA (15~20초)
| 요소 | 내용 |
|------|------|
| 장면 | 핸디버터 + 홈버터 나란히 놓인 플랫레이. 깔끔한 크림 배경. |
| 자막 | **"이번엔 정착할 것 같아요."** |
| 하단 CTA | 언버터 바디버터 · 지금 만나보기 |

### 타겟 오디언스
- 여성 20~38세
- 관심사: 스킨케어, 비건 뷰티, 자기관리, 미니멀 라이프
- 행동: 최근 30일 내 바디케어 제품 온라인 구매자

### BGM
미니멀 Lo-fi 피아노. 훅(0~3초)은 무음 또는 아주 낮은 앰비언트 → 8초부터 페이드인.

### 촬영 노트
- 앵글: 욕실 선반 정면 수평 / 텍스처 매크로 클로즈업
- 조명: 욕실 씬 쿨톤 → 언버터 씬 웜 아이보리 톤으로 전환
- 소품: 유사 바디로션 병(레이블 가리기), 언버터 제품, 키보드
- 모델: 20대 후반 여성 / 자연스러운 리액션

### KPI
| 지표 | 목표 |
|------|------|
| Hook Rate (3초) | 40%+ |
| 완주율 (15초+) | 25%+ |
| CTR | 1.5%+ |
| CPC | 300원 이하 |

## 광고 2편 — 얼굴은 챙기면서 몸 보습은 왜 안 챙김?

### 광고 제목 (내부용)
`ADV-02 | "얼굴과 몸의 불균형"`

### 훅 (0~3초)
| 요소 | 내용 |
|------|------|
| 장면 | 화장대 위 스킨케어 제품 5~6종 빽빽 → 그 옆 텅 빈 공간으로 패닝 |
| 자막/카피 | **"얼굴엔 6단계, 몸엔 0단계?"** |

### 본문 (3~15초)

| 시간 | 장면 설명 | 자막 / 나레이션 |
|------|-----------|----------------|
| 3~6초 | 얼굴에 크림 꼼꼼히 바르는 클로즈업 → 건조한 팔꿈치·종아리 | *"얼굴 보습은 매일 챙기면서,"* |
| 6~9초 | 자신의 팔을 보며 살짝 당황하는 표정 | *"몸은 언제 마지막으로 보습했어요?"* |
| 9~12초 | 홈버터를 팔에 부드럽게 펴 바르는 장면. 녹아드는 텍스처. | *"얼굴만큼, 몸도 촉촉하게."* |
| 12~14초 | 매끄러워진 팔 피부 클로즈업 + 만족 표정 | *"시어버터 15%, 언버터 바디버터."* |

### CTA (15~20초)
| 요소 | 내용 |
|------|------|
| 장면 | 화장대 스킨케어 사이에 언버터가 자연스럽게 놓인 장면 |
| 자막 | **"이제 몸도 루틴에 넣으세요."** |
| 하단 CTA | 언버터 바디버터 · 버터루틴 시작하기 |

### 타겟 오디언스
- 여성 22~40세
- 관심사: 페이셜 스킨케어, K-뷰티, 셀프케어
- 유사타겟: 기존 구매자 기반 2% LAL

### BGM
가벼운 어쿠스틱 기타 + 클라리넷. 밝고 경쾌하되 자극적이지 않음. BPM 90~100.

### 촬영 노트
- 앵글: 화장대 부감(탑뷰) + 사이드 클로즈업
- 조명: 밝은 자연광. 화장대 거울 반사 활용.
- 소품: 다양한 스킨케어 제품, 언버터 홈버터
- 편집: 얼굴 루틴 vs 몸 장면을 빠른 컷으로 대비

### KPI
| 지표 | 목표 |
|------|------|
| Hook Rate (3초) | 45%+ |
| 완주율 | 28%+ |
| CTR | 1.8%+ |
| 브랜드 인지도 리프트 | 메타 Brand Lift 조사 활용 |

## 광고 3편 — 몸에 버터를 바른다고?

### 광고 제목 (내부용)
`ADV-03 | "버터를 몸에?"`

### 훅 (0~3초)
| 요소 | 내용 |
|------|------|
| 장면 | 인물이 카메라 정면 응시. 살짝 의아한 표정. Direct-to-camera. |
| 자막/카피 | **"몸에 버터를 바른다고요?"** |

### 본문 (3~15초)

| 시간 | 장면 설명 | 자막 / 나레이션 |
|------|-----------|----------------|
| 3~5초 | 웃으며 언버터 꺼내 보여줌 | *"네, 진짜 버터예요. 몸에 쓰는."* |
| 5~8초 | 손등에 소량 올리고 문지르는 클로즈업 — 녹아드는 텍스처 | *"피부 위에서 스르르 녹아드는 느낌."* |
| 8~11초 | 바른 직후 손 흔들어 끈적임 없음 증명 | *"끈적임 없이 매끄럽고, 레몬 타르트 향이 은은하게."* |
| 11~14초 | 가방에서 핸디버터 꺼내는 → 카페에서 바르는 장면 | *"작아서 어디든 들고 다녀요."* |

### CTA (15~20초)
| 요소 | 내용 |
|------|------|
| 장면 | 인물이 카메라 보며 제품 들어 보이기 |
| 자막 | **"한번 써보면 이해해요."** |
| 하단 CTA | 핸디버터 8,900원 · 지금 써보기 |

### 타겟 오디언스
- 여성 20~35세 / 남성 보조 타겟 20~32세
- 관심사: 뷰티 트렌드, 신제품 탐색, 인플루언서 콘텐츠
- 이 편은 인지 확장용 — 넓은 타겟, CPM 최적화

### BGM
톡톡 튀는 팝 인스트루멘탈. 가볍고 재치 있는 느낌. 인물 말 타이밍에 비트 강조.

### 촬영 노트
- 앵글: 정면 미디엄샷 + 손 클로즈업
- 조명: 브라이트 스튜디오 or 창가 자연광. 밝고 생기 있게.
- 소품: 핸디버터, 토트백, 카페 소품
- 편집: 점프컷으로 빠른 전개

### KPI
| 지표 | 목표 |
|------|------|
| Hook Rate (3초) | 50%+ (궁금증 유발) |
| 완주율 | 35%+ |
| CTR | 2.0%+ |
| 공유/저장 수 | 타 편 대비 +30% |

## 광고 4편 — 줄 때마다 성공했던 선물 필수템

### 광고 제목 (내부용)
`ADV-04 | "실패 없는 선물"`

### 훅 (0~3초)
| 요소 | 내용 |
|------|------|
| 장면 | 예쁘게 포장된 언버터 선물 박스. 리본 묶인 손이 건네는 장면. |
| 자막/카피 | **"줄 때마다 좋아했던 그 선물."** |
| 연출 | 따뜻한 크림·골드 톤. 느린 줌인으로 고급스러운 첫인상. |

### 본문 (3~15초)

| 시간 | 장면 설명 | 자막 / 나레이션 |
|------|-----------|----------------|
| 3~6초 | 선물 받는 사람이 열어보며 표정 밝아짐 | *"생일, 기념일, 그냥 주고 싶어서."* |
| 6~9초 | 뚜껑 열고 향 맡는 장면. 만족 표정. | *"레몬 타르트 향, 열자마자 기분 좋아져요."* |
| 9~12초 | 손등에 발라보는 클로즈업. 녹아드는 텍스처. | *"비건 성분에 끈적임 없이 촉촉해서,"* |
| 12~14초 | 버터루틴 세트 플랫레이. 핸디+홈버터 나란히. | *"뭘 좋아할지 몰라도 이건 다 좋아해요."* |

### CTA (15~20초)
| 요소 | 내용 |
|------|------|
| 장면 | 포장된 버터루틴 세트 클로즈업 + 가격 |
| 자막 | **"버터루틴 세트 25,900원."** |
| 서브 카피 | "선물하기 딱 좋은 가격과 사이즈." |
| 하단 CTA | 선물하러 가기 |

### 타겟 오디언스
- 여성 25~42세 / 선물 구매 의향 유저
- 행동: 온라인 선물 구매 이력
- 시즌 집중: 크리스마스, 화이트데이, 빼빼로데이, 밸런타인
- 리타겟팅: 장바구니 미구매 유저

### BGM
따뜻하고 포근한 어쿠스틱. 리본 푸는 장면에서 살짝 업비트 전환.

### 촬영 노트
- 앵글: 탑뷰(플랫레이) + 45도 부감 + 클로즈업
- 조명: 따뜻한 웜 화이트. 고급스러운 느낌.
- 소품: 크림·아이보리 리본, 크라프트 박스, 드라이플라워, 원목 트레이
- 모델: 선물 주고받는 두 인물 (친구 or 커플)

### KPI
| 지표 | 목표 |
|------|------|
| Hook Rate (3초) | 42%+ |
| 완주율 | 30%+ |
| 세트 CTR | 2.2%+ |
| ROAS | 300%+ |

## 광고 5편 — 혹시 얼굴만 신경쓰다 출근하시나요?

### 광고 제목 (내부용)
`ADV-05 | "화장대 앞 바디 보습"`

### 훅 (0~3초)
| 요소 | 내용 |
|------|------|
| 장면 | 아침 화장대 앞. 얼굴 세팅하다 문득 카메라 보는 장면. |
| 자막/카피 | **"얼굴만 챙기고 출근하시나요?"** |

### 본문 (3~15초)

| 시간 | 장면 설명 | 자막 / 나레이션 |
|------|-----------|----------------|
| 3~6초 | 프라이머·선크림 꼼꼼히 → 옷 걸치며 건조한 팔 | *"세안, 스킨, 에센스, 선크림... 근데 몸은?"* |
| 6~9초 | 화장대 한 켠에서 핸디버터 꺼내는 장면 | *"이제 화장대 루틴에 하나 더."* |
| 9~12초 | 스킨케어 사이사이, 손목·팔에 가볍게 바름 | *"스킨케어 하는 1~2분 사이에,"* |
| 12~14초 | 완성된 아침 룩으로 가방 들고 나서는 뒷모습 | *"얼굴도, 몸도 준비됐어요."* |

### CTA (15~20초)
| 요소 | 내용 |
|------|------|
| 장면 | 화장대 스킨케어 사이에 자연스럽게 놓인 핸디버터 |
| 자막 | **"아침 루틴의 마지막 단계."** |
| 하단 CTA | 핸디버터 8,900원 · 버터루틴 시작하기 |

### 타겟 오디언스
- 직장인 여성 24~38세
- 행동: 출퇴근 시간대 활성 유저 (오전 7~9시, 저녁 7~9시)
- 지역: 수도권 직장 밀집 지역 우선

### BGM
상쾌한 아침 느낌. 미니멀 재즈 피아노 or 경쾌한 팝. BPM 100~115. 출근 장면에서 업비트.

### 촬영 노트
- 앵글: 화장대 거울 반사 활용 + 사이드 미디엄샷 + 클로즈업
- 조명: 아침 자연광. 따뜻한 아이보리 톤.
- 소품: 화장대, 스킨케어 제품, 핸디버터, 출근룩 의상, 가방
- 편집: 빠른 컷으로 "아침의 효율성" 느낌

### KPI
| 지표 | 목표 |
|------|------|
| Hook Rate (3초) | 45%+ |
| 완주율 | 32%+ |
| CTR | 1.8%+ |
| 구매 전환율 | 2.5%+ |

## 제작 우선순위

| 순위 | 편 | 이유 |
|-----|---|------|
| 1 | **3편 (버터를 몸에?)** | 호기심 훅 강함. 신규 인지 확장 최적. |
| 2 | **1편 (로션 정착 실패자)** | 공감형. 구매 전환 퍼널에 적합. |
| 3 | **5편 (화장대 앞)** | 타겟 명확. 직장인 아침 시간대 효율. |
| 4 | **2편 (얼굴 vs 몸)** | 브랜드 철학 전달. 중기 브랜딩. |
| 5 | **4편 (선물 필수템)** | 시즌 한정. 시즌 전 4~6주 선제 집행. |
