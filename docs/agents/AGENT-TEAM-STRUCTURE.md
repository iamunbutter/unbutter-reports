# 언버터 에이전트 팀 구조 v1

## 🎯 핵심 원칙

**하네스 엔지니어링:**
- **TASK:** 명확한 작업 단위
- **TEAM:** 역할별 팀 구성 (기획/개발/디자인)
- **LINK:** 팀 간 체인 연결
- **병렬 작업:** 각 에이전트 동시 실행 가능
- **제어 영역 내 자유 창작**

---

## 🧈 팀 구조

```
버터봇 (COO) - Opus
├─ 기획팀
│  ├─ 기획팀장 - Sonnet 4.5
│  └─ 기획팀원 - Sonnet (병렬 작업)
│
├─ 개발팀
│  ├─ 개발팀장 - Sonnet 4.5
│  └─ 개발팀원 - Sonnet/Haiku (병렬 작업)
│
└─ 디자인팀
   ├─ 디자인팀장 - Sonnet 4.5
   └─ 디자인팀원 - Sonnet (병렬 작업)
```

---

## ⚙️ 병렬 작업 시스템

### 팀 내 병렬 작업
각 팀원은 **독립적인 세션**에서 동시에 작업:

```
기획팀장 → 3명의 기획팀원에게 동시 배분
  ├─ 팀원A (세션 1): 시장 조사
  ├─ 팀원B (세션 2): 경쟁사 분석
  └─ 팀원C (세션 3): 콘텐츠 전략

→ 3개 세션 병렬 실행
→ 완료 후 팀장이 통합
```

### 팀 간 병렬 작업
의존성 없는 작업은 동시 실행:

```
버터봇 → 3개 팀에 동시 배분
  ├─ 기획팀: 기획서 작성
  ├─ 개발팀: 프로토타입 개발 (기획서 대기)
  └─ 디자인팀: 목업 제작

→ 기획팀 완료 후 개발팀 본격 시작
→ 디자인팀은 독립적으로 진행
```

## 🔄 워크플로우 (병렬 실행)

### Phase 1: TASK 접수 & 분해
```
정인 → 버터봇
"블로그 3편 써줘 + 인스타 게시물 5개 + 대시보드 업데이트"

버터봇 분석:
- TASK 1: 블로그 3편 (기획+디자인)
- TASK 2: 인스타 5개 (기획+디자인)
- TASK 3: 대시보드 (개발)

→ 3개 TASK 동시 실행 가능!
```

### Phase 2: 팀 배분 (병렬)
```
버터봇 → 3개 팀에 동시 배분

├─ 기획팀장 (세션 A)
│  "블로그 3편 + 인스타 5개 기획서"
│  └─ 팀원 3명 병렬 (블로그1, 블로그2, 블로그3)
│
├─ 개발팀장 (세션 B)
│  "대시보드 업데이트"
│  └─ 팀원 2명 병렬 (백엔드, 프론트)
│
└─ 디자인팀장 (세션 C)
   "블로그 이미지 15장 + 인스타 이미지 5장"
   └─ 팀원 3명 병렬 (블로그, 인스타, 편집)
```

### Phase 3: 작업 실행 (병렬)
```
9개 세션 동시 실행:
- 기획팀원 3명
- 개발팀원 2명
- 디자인팀원 3명
- 버터봇 (모니터링)

→ 진행 상황 실시간 대시보드 업데이트
```

### Phase 4: 통합 & 검토
```
각 팀장 → 팀원 산출물 통합
  ↓
버터봇 → 전체 통합 & 검토
  ↓
정인 → 최종 확인
```

## 🛠️ 구현 방법 (OpenClaw)

### 1. sessions_spawn으로 병렬 세션 생성
```typescript
// 기획팀원 3명 동시 실행
await Promise.all([
  sessions_spawn({
    task: "블로그 1편 기획",
    agentId: "planning-member-1",
    mode: "run"
  }),
  sessions_spawn({
    task: "블로그 2편 기획",
    agentId: "planning-member-2",
    mode: "run"
  }),
  sessions_spawn({
    task: "블로그 3편 기획",
    agentId: "planning-member-3",
    mode: "run"
  })
]);
```

### 2. sessions_yield로 결과 대기
```typescript
// 팀장이 팀원 결과 대기
sessions_yield({
  message: "팀원 3명 작업 완료 후 통합하겠습니다"
});
```

### 3. subagents로 진행 상황 모니터링
```typescript
// 버터봇이 전체 에이전트 모니터링
subagents({ action: "list" });
```
