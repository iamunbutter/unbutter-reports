# 브랜드 스토리

## 브랜드 네이밍

**unBUTTER (언버터)**

"un" + "BUTTER"

- **un:** 부정/제거의 의미
- **BUTTER:** 버터의 본질

→ "버터의 단점은 빼고, 장점만 남긴" 의미

## 브랜드 철학

**"BUTTER makes BETTER, unBUTTER"**

버터가 더 나은 것을 만든다.
언버터는 더 나은 일상을 만든다.

## 탄생 배경

### 문제 발견

```
"보습은 해야 하는데, 귀찮아."
```

- 바디로션: 끈적여서 건너뛰게 됨
- 핸드크림: 손만 바를 수 있음
- 바디오일: 무겁고 번거로움

→ 결국 "보습 루틴"이 지켜지지 않음

### 인사이트

```
"좋아야 지속된다"
```

노력해야 유지되는 루틴은 오래가지 않는다.
좋아서 쓰다 보니 매일 쓰게 되는 제품이 필요하다.

### 해결책

**버터라는 제형에 주목**

- 상온: 단단해서 휴대 편함
- 체온: 녹아서 발림성 좋음
- 보습: 크림보다 가볍지만 효과적
- 향: 베이커리 컨셉으로 차별화

## 브랜드 컨셉

### 베이커리 컨셉

**왜 베이커리인가?**

1. 버터는 베이커리의 핵심 재료
2. 베이커리는 일상의 작은 행복
3. 갓 구운 빵의 따뜻함과 포근함
4. 감각적이고 트렌디한 이미지

### 비주얼 아이덴티티

**브랜드 컬러:**
- #fff1b5 (Soft Butter) - 부드러운 노란빛
- #43302e (Cocoa Latte) - 따뜻한 브라운

**로고:**
- 변형 로고 메인 사용
- 베이커리 감성 극대화
- 패키지/SNS에서 강한 인상

## 제품 네이밍 전략

### 버터루틴 시리즈

**"매일이 설레는 버터루틴"**

- **핸디버터:** 언제 어디서나
- **홈버터:** 집에서 충분히
- **버터루틴 세트:** 밖에서도 집에서도

### 향 네이밍

**Lemon Tart (레몬 타르트)**

- 베이커리 컨셉에 부합
- 구체적이고 연상 쉬움
- 트렌디하고 감각적

(향후 확장: 바닐라 타르트, 얼그레이 타르트 등)

## 고객과의 약속

### 1. 정직함
- 과장하지 않는다
- 성분 투명하게 공개
- 후기 조작하지 않는다

### 2. 일관성
- 브랜드 정체성 유지
- 메시지 통일
- 품질 안정성

### 3. 공감
- 고객의 불편함 이해
- 실제 사용 장면 반영
- 리얼한 후기 존중

## 브랜드 슬로건

**서브:**
- "크림보다 부드럽고, 오일보다 가벼운 버터"
- "좋아서 지켜지는 매일의 루틴"
- "버터 같은데 버겁지 않은"

- 트렌디한

## 차별화 포인트

### vs 더바디샵, 러쉬
- 우리: 데일리 루틴 중심
- 경쟁사: 특별한 날 케어

### vs 탬버린즈, 논픽션
- 우리: 베이커리 컨셉으로 차별화
- 경쟁사: 미니멀/모던

### vs 휩드
- 우리: 30g 핸디버터로 휴대성 강조
- 경쟁사: 대용량 위주

## 미래 비전

**1차 목표 (2026-2029)**
국내 온라인 바디케어 시장 점유율 5% (50억)

**2차 목표 (2030~)**
- 국내 점유율 10% (100억)
- 글로벌 진출 (100억)
- 바디케어 토탈 브랜드 확장

**확장 가능성:**
- 향 라인 확장 (바닐라, 얼그레이 등)
- 용량 다양화
- 제형 확장 (바디워시, 로션 등)
- 굿즈/컬래버레이션
