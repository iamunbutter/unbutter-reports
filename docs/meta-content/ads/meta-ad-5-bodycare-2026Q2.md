# 광고 1편 — 바디로션 정착 못하는 이유

## 🎯 영상 목표
- **문제장면 주제:** "또 샀다가 또 방치" 루프에 공감시키기
- **타겟:** 바디로션 3개 이상 쌓아두고 다 반만 쓴 2030 여성

## 🚨 문제장면 (0~6초)
- **상황:** 욕실 선반에 반쯤 쓰다 만 바디로션 4~5개 줄지어 있음
- **행동:** 하나 집어서 펌프 눌렀는데 안 나옴. 다른 거 집었는데 뚜껑 열기 귀찮아서 그냥 내려놓음. 안 바름.
- **감정:** 귀찮음 + 자책 + 체념
- **음성:** (혼잣말) "...그냥 안 바르지 뭐."
- **자막:**
  - "바디로션 무덤 보유자 🙋"
  - "왜 살 때마다 결국 이렇게 됨..?"
  - "로션 정착 실패 N회차"
- **효과:** 선반 클로즈업 → 빠른 컷으로 로션 하나씩 줌인 → 내려놓는 손 슬로우모션

## 💡 전환장면 (6~12초)
- **상황:** 릴스 보다가 손에 뭔가 바르는 영상에서 엄지 멈춤
- **행동:** 댓글 확인 → "언버터 핸디버터임 나 세 통째" → 링크 클릭
- **감정:** "이게 다른 건가?" 하는 작은 호기심
- **음성:** "어? 이건 좀 달라 보이는데."
- **자막:**
  - "또 속으면 어떡하지 하면서도"
  - "결국 클릭함 ㅋㅋ"

## ✨ 이상장면 (12~20초)
- **상황:** 침대에 누워서 핸디버터 한 번 쭉 발라봄
- **행동:** 5초 만에 바르고 아무 일 없다는 듯이 폰 잡음. 손 비벼보고 "오?" 표정.
- **감정:** "어 이거 진짜 됨?" 조용한 놀람
- **음성:** "이게 왜 이렇게 흡수가 빨라?"
- **자막:**
  - "5초 바르고 폰 잡았는데 안 묻음"
  - "이래서 정착하는 거였구나"
  - "로션 무덤 처분 결심 🫡"
- **CTA:** 핸디버터 8,900원 — 링크 바이오에

# 광고 2편 — 얼굴은 챙기면서 몸 보습은 왜 안 챙김?

## 🎯 영상 목표
- **문제장면 주제:** 스킨케어 진심이지만 팔뚝은 당김 — 괴리감 공략
- **타겟:** 얼굴 루틴 10단계인데 바디케어는 0단계인 사람

## 🚨 문제장면 (0~6초)
- **상황:** 세면대 앞에서 얼굴에 세럼, 크림, 선크림까지 꼼꼼하게 레이어링 중
- **행동:** 얼굴 다 바르고 흡족한 표정 → 무의식적으로 팔뚝 긁음 → 흠칫
- **감정:** 찰나의 자각 → "어 나 팔 엄청 당기는데?"
- **음성:** "잠깐, 팔 왜 이렇게 당겨?"
- **자막:**
  - "얼굴: 토너→세럼→크림→선크림 풀루틴"
  - "팔뚝: 🦎"
  - "이 분리가 좀 웃기긴 함"
- **효과:** 얼굴 케어 빠른 컷 몽타주 → 팔 긁는 장면에서 갑자기 브레이크 컷

## 💡 전환장면 (6~12초)
- **상황:** 회사 동료가 팔뚝에 뭔가 쓱 바르는 거 보임
- **행동:** "야 그거 뭐야?" → 동료가 작은 통 건네줌 → 한 번 발라봄
- **감정:** 끈적 안 하고 바로 흡수됨 → 표정 변함
- **음성:** "어 이거 이상하다, 왜 이렇게 빨리 스며들어?"
- **자막:**
  - "회사에서 옆자리한테 뺏어 바른 거"
  - "근데 진짜 달랐음"

## ✨ 이상장면 (12~20초)
- **상황:** 아침 스킨케어 마지막에 팔에도 핸디버터 쭉 바르는 장면
- **행동:** 얼굴이랑 같은 흐름으로 팔에 바름. 거울 보면서 팔 쭉 뻗어봄.
- **감정:** 완성된 느낌. "이제 좀 제대로 챙기는 것 같다."
- **음성:** "얼굴이랑 팔이랑 같이 챙겨야지, 왜 이걸 몰랐지?"
- **자막:**
  - "얼굴만 빛나면 반쪽짜리임"
  - "루틴 마지막 칸 드디어 채움 ✨"
  - "팔도 광내는 중"
- **CTA:** 바디 루틴 시작템 — 링크 바이오에

# 광고 3편 — 몸에 버터를 바른다고?

## 🎯 영상 목표
- **문제장면 주제:** "바디버터? 기름지잖아" 선입견 → 직접 발라보고 뒤집힘
- **타겟:** 바디버터 써보고 싶었지만 끈적일 것 같아서 안 써본 사람

## 🚨 문제장면 (0~6초)
- **상황:** 친구가 "이거 진짜 좋아" 하면서 바디버터 내밀음
- **행동:** 손에 올려놓고 의심스럽게 쳐다봄. 손가락으로 찍어봤다가 얼굴 찡그림.
- **감정:** "이거 바르면 하루 종일 끈적거리겠다" 확신
- **음성:** "야 이거 완전 기름 아니야? 나 이런 거 못 써."
- **자막:**
  - "버터를 몸에 바른다고..?"
  - "기름 광 날 것 같은 불길한 예감"
  - "손에 올리자마자 후회"
- **효과:** 버터 질감 클로즈업 → 의심 표정 줌인 → 슬로우모션으로 손가락 찍는 장면

## 💡 전환장면 (6~12초)
- **상황:** 친구가 "그냥 한 번만 발라봐" 하면서 반강제로 팔에 발라줌
- **행동:** 인상 쓰면서 발렸는데 → 5초 후 손으로 비벼봄 → 표정이 묘하게 바뀜
- **감정:** "어? 이거 스며들었어?" 당황 + 배신감(?)
- **음성:** "잠깐, 이거 왜 안 끈적여?"
- **자막:**
  - "5초 뒤"
  - "...어?"
  - "배신당한 선입견"

## ✨ 이상장면 (12~20초)
- **상황:** 혼자 집에서 침대 위에 앉아 바디버터 직접 꺼내서 바르는 중
- **행동:** 손에 떠서 양 팔뚝에 쭉 바름. 슥 올리는 정도. 팔 쭉 뻗어보고 흐뭇.
- **감정:** "이거 왜 이렇게 바르기 쉬워?" 가벼운 충격
- **음성:** "진짜 버터인데 왜 버터 같지가 않아..."
- **자막:**
  - "버터가 이렇게 흡수가 되네"
  - "선입견 박살난 날"
  - "나 어떡해 이거 계속 살 것 같아 🧈"
- **CTA:** 언버터 핸디버터 — 직접 발라봐야 앎

# 광고 4편 — 줄 때마다 성공했던 선물 필수템

## 🎯 영상 목표
- **문제장면 주제:** 선물 고르는 고통 → 언버터로 해방 → 받은 사람 반응까지
- **타겟:** 생일/기념일 선물 고를 때마다 멘붕 오는 사람

## 🚨 문제장면 (0~6초)
- **상황:** 카톡에서 친구 생일 알림 뜸. 쇼핑앱 켜서 "선물" 검색.
- **행동:** 스크롤 내리다 올리다 반복. "이건 너무 흔하고", "이건 너무 비싸고". 탭 5개 열림.
- **감정:** 선물 고르는 게 더 힘듦 → 지침
- **음성:** "아 진짜 뭐 사줘야 하지. 또 상품권이냐."
- **자막:**
  - "선물 고르는 시간 > 일 하는 시간"
  - "취향도 모르고 뭘 사줘.."
  - "탭 5개 열어놓고 결국 제자리"
- **효과:** 쇼핑앱 스크롤 화면 빠른 컷 → 고민 표정 클로즈업

## 💡 전환장면 (6~12초)
- **상황:** 다른 친구한테 카톡 옴. "나 생일 선물 언버터 보내줬는데 얘 완전 난리남ㅋㅋ"
- **행동:** 캡처 보여줌 — 받은 친구 카톡: "이거 뭐야!!!! 개좋아 어디서 났어"
- **감정:** "어 이거 정답이네?" 확신
- **음성:** "야 이거 받은 사람이 저렇게 반응하면 답이지."
- **자막:**
  - "선물받은 친구 반응"
  - "이거 뭐야!!!!! 개좋아"
  - "이 반응이면 됐지 뭐 🎯"

## ✨ 이상장면 (12~20초)
- **상황:** 친구 생일에 버터루틴 세트 선물로 줌. 친구가 뜯으며 반응.
- **행동:** 열자마자 "오 이게 뭐야?" → 바로 냄새 맡음 → 눈 커짐 → "향 미쳤다"
- **감정:** 주는 사람도 뿌듯. "역시 이거지."
- **음성:** (친구) "야 이거 어디서 났어, 모르는 브랜드인데 좋다"
- **자막:**
  - "받자마자 냄새부터 맡는 친구"
  - "줄 때마다 이 반응임"
  - "선물 고민 이제 끝 🎁"
- **CTA:** 버터루틴 세트 25,900원 — 링크 바이오에

# 광고 5편 — 혹시 얼굴만 신경쓰다 출근하시나요?

## 🎯 영상 목표
- **문제장면 주제:** 아침 준비에서 몸은 완전히 스킵 → 직장인 공감 극대화
- **타겟:** 아침에 시간 없어서 바디케어 포기한 직장인

## 🚨 문제장면 (0~6초)
- **상황:** 아침 7시. 알람 끄자마자 세면대 앞. 클렌징→토너→세럼→크림→선크림 빠르게 컷 전환.
- **행동:** 얼굴 다 하고 옷 입다가 팔뚝 당기는 느낌. 긁음. 시간 없어서 그냥 나감.
- **감정:** 체념. "오늘도 그냥 가야지."
- **음성:** "아 팔 당기는데... 시간 없으니까 그냥 가자."
- **자막:**
  - "직장인 아침 루틴: 얼굴 10분 / 몸 0초"
  - "오늘도 팔뚝은 희생됨"
  - "매일 이러면서 매일 후회함 ㅋ"
- **효과:** 아침 스킨케어 빠른 컷 몽타주 → 팔 긁는 브레이크 → 시계 클로즈업

## 💡 전환장면 (6~12초)
- **상황:** 지하철에서 맞은편 사람이 작은 통 꺼내서 손이랑 팔에 쭉 바름
- **행동:** 쳐다봄 → 바른 팔이 끈적해 보이지 않음 → 핸드폰 들고 검색 시작
- **감정:** "저거 지하철에서 바를 수 있는 거야?" 신기함
- **음성:** "지하철에서 바르는데 손 안 끈적여?"
- **자막:**
  - "지하철에서 저러는 사람 처음 봄"
  - "근데 왜 자연스럽지"
  - "나도 저거 하면 되는 거 아님?"

## ✨ 이상장면 (12~20초)
- **상황:** 다음 날 아침. 지하철 안에서 팔뚝에 쭉 바름. 출근 완료.
- **행동:** 10초도 안 걸리게 바르고 가방에 넣음. 팔 뻗어보고 만족한 표정으로 이어폰 꽂음.
- **감정:** "이거 생각보다 너무 쉽잖아." 조용한 뿌듯함
- **음성:** "이거 왜 이렇게 간단해. 이걸 왜 지금 알았지?"
- **자막:**
  - "아침 루틴 마지막 칸 드디어 채움"
  - "지하철에서 10초면 끝"
  - "얼굴이랑 팔이랑 같이 출근하는 중 🫶"
- **CTA:** 출근길 보습 완성 — 핸디버터 링크 바이오에
