핵심은 "내 문제를 해결할 수 있겠다!"는 깨달음입니다.

</aside>

<aside>
📍

**상황** (어디서)

화장대 앞

</aside>

<aside>
🎬

**행동** (뭘 해)

- 홈버터, 핸디버터 나란히 보여주기
- 홈버터는 집에 두고, 핸디버터만 가방에 쏙
</aside>

<aside>
🎬

**행동** (뭘 해)

- 홈버터는 집에 두는 모습 (2초)
- 핸디버터만 가방에 쏙 넣기 (2초)
- 두 개 나란히 보여주며 비교 (2초)
</aside>

<aside>
🤩

**감정** (표정)

깨달음

</aside>

<aside>
💬

**음성**

언버터 바디버터는 손부터 팔다리까지 하나로 전부 케어 가능하고,

홈용량은 집에 두고 가방엔 핸디버터 하나만 넣어 다니면 끝!

**자막**

시어버터 15% / 끈적임 없이 흡수 / 2type 용량

</aside>

<aside>
💬

**음성**

이건 시어버터 15% 함유에 끈적임 없이 흡수되는데,

홈케어용 따로 있어서 집에 두고 쓰고

휴대용만 가방에 쏙 넣으면 되니까 완전 편해요

**자막**

시어버터 15% / 끈적임 없이 흡수 / 2type 용량

홈버터 100g → 집에 두고,

핸디버터 30g → 가방에 쏙!

</aside>

# ✨ 이상장면

<aside>
✨

**시청자가 "와 나도 저렇게 되고 싶다" 하면서 기대감을 갖는 장면**

문제가 해결된 후의 만족감! 고통은 없습니다.

</aside>

<aside>
📍

**상황** (어디서)

집 - 침대 / 화장대

밖 - 사무실 / 카페 / 지하철

</aside>

<aside>
🎬

**행동** (뭘 해)

- 집에서 홈버터 바르기
- 가벼운 가방에 핸디버터만 쏙
- 가방에서 바로 꺼내 바르기
- 향 맡으며 만족

**[Before vs After]**

- 가방 속: 전에는 5개 보습템 → 지금은 핸디버터 1개
- 가벼워진 가방을 들어보이는 모습
</aside>

<aside>
🎬

**행동** (뭘 해)

- 집에서 홈버터로 팔, 다리 넓게 바르기 (3초)
- 가벼운 가방 들며 만족스러운 표정 (2초)
- 사무실에서 가방에서 바로 꺼내 바르기 (3초)
- "찾기 쉬워!" 만족하는 모습 (2초)
- 향 맡으며 만족하는 표정 (2초)
- 제품 클로즈업 + 브랜드명 (4초)
</aside>
