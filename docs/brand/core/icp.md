# ICP (Ideal Customer Profile)

## 핵심 고객 정의

**타겟:** 20-30대 직장인 여성

**페르소나:**
- 이름: 박서윤
- 나이: 29세
- 직업: IT 기업 콘텐츠 디자이너
- 월소득: 250-300만원

## 고객 특성

### 라이프스타일
- 출퇴근 지하철 이용
- 에어컨/히터에 장시간 노출
- 루틴 관리 중요시
- SNS 활발 (인스타, 블로그)

### 가치관
- "좋아서 쓰게 되는 제품" 선호
- 패키지 디자인 중요
- 제품 = 나의 이미지
- 검증된 제품만 사용

### 구매 행동
- 인스타 → 블로그 후기 → 구매
- 광고 티 나는 콘텐츠 기피
- 한 번 마음에 들면 반복 구매
- 주변에 적극 추천

## Pain Points

1. 기존 바디로션: 끈적이거나 금방 건조
2. 향수: 무겁고 매일 쓰기 부담
3. 패키지: 촌스러워 화장대 인테리어 방해
4. 휴대성: 큰 용량은 부담, 작은 용량은 아까움

## 니즈

1. 끈적임 없는 보습
2. 은은한 시그니처 향
3. 예쁜 패키지 (데스크테리어)
4. 추천 자랑스러운 브랜드

## 구매 결정 요소

1. **사용감** - 끈적이지 않고 빠른 흡수
2. **향** - 은은한 레몬 버터향
3. **편의성** - 언제 어디서나 사용
4. **디자인** - 화장대/데스크 감성
5. **소장 가치** - 나만 알고 싶지만 보여주고 싶은

## 채널

### 발견 경로
- 인스타그램 (릴스/피드)
- 네이버 블로그
- 지인/인플루언서 추천

### 구매 전환 트리거
- "끈적이지 않는다" 후기
- "향이 은은하다" 후기
- 예쁜 패키지 사진
- 사용 장면 영상

### 이탈 사유
- 가격대 부담
- 향이 취향 아님
- 차별성 불명확
