# 저성과 패턴 (What Doesn't Work)

_실패한 콘텐츠/전략 기록 (같은 실수 반복 방지)_

## 아직 데이터 없음

런칭 후 저성과 콘텐츠가 나오면 여기에 기록합니다.

### 기록 형식

```markdown
## [날짜] [콘텐츠명]

### 성과
- 조회수/도달: X (목표 대비 Y% 미달)
- 참여율: X% (목표 대비 낮음)
- 전환: X건 (기대 이하)

### 왜 안 됐나?
- ...

### 배운 점
- ...

### 다음엔 이렇게
- ...
```
