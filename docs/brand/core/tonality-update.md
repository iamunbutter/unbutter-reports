# 톤앤매너 업데이트 (블로그 톤 반영)

## 채널별 톤 수정

### 인스타그램 피드
- **존댓말 사용** (친근한 존댓말)
- 주인장 → 버터팬 관계
- 간결하고 감각적
- 이모지: 🧈, 💛, ✨
- 2-3문장으로 핵심 전달

**예시:**
```
드디어 샘플 확정됐어요 🧈

몇 달 고민한 끝에 결정한
레몬 타르트 향!

🍋 바를 때: 싱그러운 레몬 제스트
🧈 10분 후: 고소한 버터 크러스트  
☁️ 1시간 후: 포근한 머스크 잔향

버터팬들은 어떤 향이 궁금하신가요? 🤔
```

---

### 네이버 블로그

**톤:** 주인장의 일기 느낌
- 친근한 존댓말 ("~해요", "~하세요")
- 솔직하고 진솔하게
- 과정과 고민 공유
- 스토리텔링 중심

**말투 특징:**
- "~지요?", "~하네요", "~더라고요"
- "짠!", "그만큼 진심이고 간절하다는 의미이기도 하겠죠?"
- 괄호로 속마음 표현
- 이모지: 🧈 💛 🤎

**예시 (실제 블로그 발췌):**
```
"두 달 간 7개가 넘는 샘플을 반려시키며
머릿속에 있는 제품을 실물로 짠! 구현해 내는 것이 
얼마나 어려운 일인지 새삼 깨달았습니다"

"제법 버터 같지요?"

"많이 많이 괴롭혀주세요"

"늦지 않게 또 글 적을게요 안녕 💛🤎"
```

## 핵심 변경사항

**기존:** "불필요한 존댓말보다 친근한 반말"
**수정:** "친근한 존댓말 사용 (주인장 ↔ 버터팬)"

**이유:**
- 커뮤니티 느낌 강화
- 팬덤 형성에 유리
- 롬앤 같은 성공 브랜드 벤치마크
